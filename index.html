como usar

Guia completo — 6 passos

Qualquer pessoa consegue usar. Não precisa instalar nada — só o Discord aberto.

01
Habilite o DevTools (se necessário)
O Discord Stable bloqueia o console por padrão. Se F12 não abrir, habilite:
Windows
  • Abra: %appdata%/discord/settings.json
  • Adicione e reinicie:
  • "DANGEROUS_ENABLE_DEVTOOLS_ONLY_ENABLE_IF_YOU_KNOW_WHAT_YOURE_DOING": true
Linux / macOS
  • Abra: ~/.config/discord/settings.json
  • Adicione a mesma linha e reinicie
Alternativa
Use Discord PTB ou Canary — DevTools habilitado por padrão. Funciona igual ao Stable.
02
Inscreva-se nas missões
Discord → aba Missões → inscreva-se. Missões de jogos exigem o jogo instalado e vinculado à conta.
03
Abra o console do Discord
Windows / Linux
  • F12
  • Ctrl+Shift+I
macOS
  • Cmd+Opt+I
Dica
Se aparecer "allow pasting", digite exatamente isso e Enter antes de colar.
04
Clique na aba Console
No DevTools que abriu, clique em Console no topo.
05
Cole e pressione Enter
Clique em Copiar Script acima, cole com Ctrl+V e pressione Enter. Para parar: yKStop() · Status: yKStatus()
06
Colete as recompensas
Após terminar, abra a aba Missões no Discord e colete manualmente. A coleta automática não funciona por design da plataforma.

Perguntas frequentes

Não. JavaScript puro que roda dentro do console do seu próprio Discord. Não baixa nada, não instala nada e não acessa arquivos do computador. Quando você fecha o Discord, o script para.
O script é distribuído protegido para preservar a autoria de yKauttiz. É uma prática comum de proteção de trabalho. O Discord executa normalmente e os logs coloridos aparecem como no preview acima.
O servidor é o canal oficial de suporte e atualizações. O script verifica sua presença automaticamente. Se não conseguir entrar, acesse discord.gg/G3wrqyJQrC manualmente e rode novamente.
USE POR SUA CONTA E RISCO. O script usa APIs internas não oficiais do Discord. yKauttiz não se responsabiliza por banimentos. Para mais segurança: use SAFE_MODE: true ou uma conta secundária.
WATCH_VIDEO e PLAY_ACTIVITY funcionam em ambos.
PLAY_ON_DESKTOP e STREAM_ON_DESKTOP exigem apenas o Desktop App — detectados e pulados automaticamente no navegador.
Com CFG.PARALLEL_VIDEO = true, o script executa até 2 vídeos simultaneamente com um jitter de 800ms-2500ms entre eles para evitar rate limiting sincronizado. Experimental — use com cautela em contas principais. Para mais segurança combine com SAFE_MODE: true.
yKStop() — para graciosamente (substitui yKauttizStop, mas o alias ainda funciona)
yKStatus() — status em tempo real sem parar o script
yKLog() — últimas 20 entradas do log
yKMetrics() — métricas por tipo de missão

O Discord processa recompensas com validação anti-fraude interna. A coleta automática não funciona por design da plataforma. Após o script terminar, basta abrir a aba Missões — as recompensas estão lá.
